ARQUIVOS DE SOM PARA NOTIFICAÇÕES
==================================

Coloque os seguintes arquivos .wav nesta pasta:

1. new_chamado.wav    - Som para novo chamado aberto
2. resposta.wav       - Som para nova resposta recebida
3. alerta.wav         - Som para SLA crítico
4. notification.wav   - Som genérico (fallback)

Se os arquivos não existirem, o sistema usará
o beep padrão do Windows.

RECOMENDAÇÕES:
- Use arquivos .wav curtos (1-3 segundos)
- Volume moderado para não incomodar
- Sons distintos para cada tipo de notificação
